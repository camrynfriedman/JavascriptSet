Game of Set

How to Play

In the game of Set, 12 cards are dealt face up on the table.

Each card has four attributes

  • Shape: Squiggle, Oval, Diamond
  • Color: Red, Purple, Green
  • Number: 1, 2, 3
  • Shading: Filled, Striped, Open

The object of the game is to get a Set. In this case, a Set consists of 3 cards where each of the card's features are the SAME on each card, or DIFFERENT on each card.

When you identify a correct set, the cards will be removed from the table and three new ones will appear.

There's a chance that you are unable to identify any sets on the table. In this case, you can choose to add three new cards to the table.
